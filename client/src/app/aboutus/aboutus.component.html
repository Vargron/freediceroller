D-Roll.com

D-roll.com is an attempt to make it easier for tabletop rpg players to manage characters and play when they don't have dice handy. Most other elements of a game can eaily be run from a laptop or phone. This is an attempt to make those games more portable and accessible.

Technologies used
  • AngularJs
  • Express
  • Node
  • MongoDB
  • Twitter Bootstrap

This is a full MEAN application, my goal was to make an intuivie system that would utilize client side processing as much as possible. The code can be seen on my github.
