Freediceroller.com

Freediceroller.com is an attempt to make it easier for tabletop rpg players to manage characters and play when they don't have dice handy. Most other elements of a game can eaily be run from a laptop, this is an attempt to make table-top games more portable.

Technologies used
  • AngularJs
  • Express
  • Node
  • MongoDB
  • Twitter Bootstrap
